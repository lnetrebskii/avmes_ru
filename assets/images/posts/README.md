# Изображения для постов блога

Эта папка содержит изображения для постов блога.

## Структура

Все изображения постов должны находиться в этой папке: `assets/images/posts/`

## Именование файлов

Рекомендуется использовать понятные имена, соответствующие слагу поста:

- `pochemu-avtomatizaciya-vazhna.jpg` - для поста "Почему офис должен окупать себя автоматически"
- `kabinet-samara-storytelling.jpg` - для поста про кабинет в Самаре
- `psiholog-anna-avtomatizaciya.jpg` - для поста про психолога Анну

## Подключение к посту

В front matter каждого поста нужно добавить поле `image`:

```yaml
---
layout: post
title: "Название поста"
image: /assets/images/posts/имя-файла.jpg
---
```

## Форматы

Поддерживаемые форматы: `.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`

## Размеры

Рекомендуемые размеры для изображений постов:
- Ширина: 1200-1600px
- Высота: пропорционально (обычно 4:3 или 16:9)
- Размер файла: до 500KB (для быстрой загрузки)
